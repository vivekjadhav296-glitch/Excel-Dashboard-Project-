**Airline-Excel-Dashboard**
This is a Dashboard of an Airline made in Microsoft Excel.

**About Airline Dataset**
The Dataset is taken from Kaggle. Here is the link: https://www.kaggle.com/datasets/iamsouravbanerjee/airline-dataset/data. The Dataset contains data related to the Airline. There are 15 columns which include: Passenger ID, First Name, Last Name, Gender, Age, Nationality, Airport Name, Airport Country Code, Country Name, Airport Continent, Continents, Departure Date, Arrival Airport, Pilot Name and Flight Status. There are 98620 rows in this dataset.

**Overall Process:**
Download the Dataset From the Kaggle.
Open the dataset in Excel.
Analyze the whole data with an eye and apply a filter to remove any wrong value or empty row.
Using Pivot Tables and Pivot Charts, the Airline Dashboard in Excel is Ready.
Format the Dashboard by color, and font and add an image to make it more attractive.
Add a slicer and connect the slicer to all pivot tables.
**KPI's:**
Total Passengers Travelled By Top 10 Nationality represented with the help of Bar Graph.
Most Continent visited by Passengers are shown in the Bar Graph.
Flight Status: Cancelled, Delayed and On-Time can be seen in Line Graph.
Total Traveller By Gender can be represented on a Pie Chart.
Top 5 Nationality Distributions of Passengers are also shown in the Pie Chart.
